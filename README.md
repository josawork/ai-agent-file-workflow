# 双AI文件协作Agent Demo
基于本地文件通信的多Agent协同工作流演示项目

## 项目介绍
本项目模拟企业AI原生工作流，拆分双AI角色分工：
- 规划Agent：DeepSeek V4，负责任务拆解、清单下发、成果验收
- 执行Agent：Claude，负责读取任务、落地执行、状态更新、结果回写

采用`next_action.md`作为唯一通信载体，实现离线轻量化的任务分发自动化闭环。

## 技术栈
- Python：脚本开发、文件读写、自动化逻辑实现
- Markdown：充当AI间任务通信、协议定义载体
- Config配置化：分离项目参数与业务代码
- 本地日志系统：完整记录每一次项目运行流程
